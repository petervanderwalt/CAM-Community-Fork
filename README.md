# CAM Community Fork

> **⚠️ Not affiliated with, endorsed by, or supported by OpenBuilds.**
>
> The original OpenBuilds business has closed down. This repository is a **community fork** of the last available source code, maintained independently to keep the project alive, fix bugs, and provide a working CAM solution for users. All product names, trademarks, and brands are the property of their respective owners.

**CAM** is a browser-based tool for converting DXF, SVG, Bitmap (BMP, JPG, PNG, GIF) and Gerber X274 files to G-Code for CNC machines, lasers, plasma cutters, drag knives, and pen plotters.
